Execute()

Executes the boolean command by resolving and evaluating its expression.

  • If the expression is empty, the value defaults to false.
  • If the expression is true or false, the value is set accordingly.
  • If the expression is numeric, non-zero values evaluate to true.
  • All other values result in an exception.
