Interface IAppCanvas

Namespace
BOOSEide
Assembly
BOOSEide.dll

Application-specific canvas interface for use with AppCanvasCommand commands. Mirrors a standard ICanvas but is fully compatible with the App* command classes. Provides drawing operations, shape creation, text output, and canvas state management.
