BOOSEide Documentation

Welcome to the BOOSEide IDE documentation. This includes both the user guides and the full API reference for developers.

Getting Started

  1. Download and install BOOSEide.
  2. Open the IDE.
  3. Use the input box to run BOOSE commands such as moveto, circle, rect, and others.
  4. The canvas will update as commands are executed.

Commands Overview

The following are some of the key commands available in BOOSEide:

  • moveto x,y – Moves the pen to a specific coordinate.
  • circle radius – Draws a circle at the current pen position.
  • rect width,height – Draws a rectangle from the current pen position.
  • pen r,g,b – Changes the pen color.
  • write "text" – Writes text at the current pen position.

For full details, see the API Reference below.

Notes

  • All coordinates and sizes are integers.
  • Negative values will throw exceptions.
  • Commands are case-insensitive.
